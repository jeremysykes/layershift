Ready-to-embed components that make video feel alive on any website. No framework dependencies. One script tag. Works in plain HTML, React, Vue, Svelte, Angular, WordPress — anywhere.

Depth Parallax

Adds depth-aware parallax motion to any video. Near objects move more than far objects, creating a convincing 3D effect from a single 2D video. Move your mouse over the demo below.

One script tag. One element.

<!-- Add the script --> <script src="https://yourdomain.com/components/depth-parallax.js"></script> <!-- Use the element --> <depth-parallax src="video.mp4" depth-src="depth-data.bin" depth-meta="depth-meta.json" ></depth-parallax>
<script src="https://yourdomain.com/components/depth-parallax.js"></script> <depth-parallax src="video.mp4" depth-src="depth-data.bin" depth-meta="depth-meta.json" ></depth-parallax>
import { DepthParallax } from 'depth-parallax/react' export default function Hero() { return ( <DepthParallax src="video.mp4" depthSrc="depth-data.bin" depthMeta="depth-meta.json" /> ) }
<template> <DepthParallax src="video.mp4" depth-src="depth-data.bin" depth-meta="depth-meta.json" /> </template> <script setup> import DepthParallax from 'depth-parallax/vue' </script>
<script> import DepthParallax from 'depth-parallax/svelte' </script> <DepthParallax src="video.mp4" depthSrc="depth-data.bin" depthMeta="depth-meta.json" />
import { DepthParallaxComponent } from 'depth-parallax/angular' @Component({ imports: [DepthParallaxComponent], template: ` <app-depth-parallax src="video.mp4" depthSrc="depth-data.bin" depthMeta="depth-meta.json" /> ` }) export class HeroComponent {}

Configuration

Attribute Type Default Description
src string — Video file URL (required)
depth-src string — Precomputed depth binary URL (required)
depth-meta string — Depth metadata JSON URL (required)
parallax-x number 0.4 Horizontal parallax intensity
parallax-y number 1.0 Vertical parallax intensity
parallax-max number 30 Max pixel offset for nearest layer
overscan number 0.05 Extra padding to prevent edge reveal
autoplay boolean true Auto-play on element mount
loop boolean true Loop video playback
muted boolean true Muted playback (required for autoplay)

Prepare your video

Videos need precomputed depth data. Run the CLI tool on your video once — it produces the depth binary and metadata files. Then embed.

# Install npm install depth-aware-parallax-video # Generate depth data npm run precompute -- your-video.mp4
